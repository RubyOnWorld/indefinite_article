# indefinite_article

Adds indefinite article methods to String and Symbol so that you can do:

    >> 'apple'.indefinite_article
    => 'an'
    >> 'banana'.with_indefinite_article
    => 'a banana'
    >> 'university'.indefinitize
    => 'a university'

## Installation

    $ gem install indefinite_article
